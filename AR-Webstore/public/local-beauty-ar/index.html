About This Page

This is a local version of the Beauty AR Experience that works without requiring external server connections. Due to browser security restrictions, camera access for AR features requires either:

  • A secure connection (HTTPS)
  • A localhost connection

Troubleshooting Steps:

  1. Allow Camera Access - When prompted, make sure to allow camera access
  2. Use Chrome or Firefox - These browsers have better support for camera API
  3. Try Local Server - Run a local server for better browser compatibility
  4. Check File Paths - Ensure the beauty-web directory exists in both locations
  5. Internet Connection - For hosted demos, ensure you have internet connectivity
